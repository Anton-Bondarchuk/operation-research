# Объяснение решения задачи 20.4

## Описание овражной функции

Функция $f(x,y) = (9x+8y-5)^4+(5x+2y-1)^4$ является типичным представителем класса овражных функций. Такие функции характеризуются:
- Наличием длинных узких "оврагов" или "долин" с очень пологим дном
- Крутыми склонами в одних направлениях и пологими в других
- Высокой степенью вытянутости линий уровня

В данном случае функция состоит из двух членов, возведенных в 4-ю степень, что создает резкие перепады градиента и формирует сложный овражный рельеф поверхности.

## Методы оптимизации и их результаты

### 1. Метод Нелдера-Мида для овражной функции

**Принцип работы:** Метод прямого поиска без использования производных, основанный на геометрических операциях с симплексом.

**Результаты:**
- Потребовалось значительное количество итераций и вычислений функции
- Метод медленно продвигается вдоль оврага
- Путь оптимизации на контурном графике показывает сложности с определением направления вдоль дна оврага

**Комментарий:** Метод Нелдера-Мида не очень хорошо подходит для овражных функций, так как ему сложно адаптировать форму симплекса к сильно вытянутым линиям уровня функции. Метод тратит много вычислений на "зигзагообразное" продвижение вдоль оврага.

### 2. Метод Ньютона для овражной функции

**Принцип работы:** Использует информацию о градиенте и аппроксимацию гессиана для выбора направления поиска.

**Результаты:**
- Потребовалось значительно меньше итераций и вычислений функции по сравнению с методом Нелдера-Мида
- Лучше справляется с движением вдоль оврага
- Путь оптимизации более прямолинеен и эффективен

**Комментарий:** Метод Ньютона лучше справляется с овражными функциями благодаря использованию информации о кривизне функции, что позволяет ему быстрее определять направление вдоль дна оврага.

## Анализ эффективности методов для овражной функции

### Проблемы при оптимизации овражных функций:

1. **Масштабные различия градиента** в разных направлениях:
   - Вдоль оврага градиент очень мал
   - Поперек оврага градиент очень большой

2. **Плохая обусловленность задачи** - отношение максимального и минимального собственных значений гессиана велико, что приводит к численной нестабильности.

3. **Зигзагообразное поведение** методов оптимизации - многие методы совершают множество маленьких шагов, попеременно ударяясь о "стенки" оврага.

### Сравнение методов:

1. **Метод Нелдера-Мида**:
   - Преимущества: не требует вычисления производных, более устойчив к шумам
   - Недостатки: медленная сходимость на овражных функциях, требует большого числа вычислений

2. **Метод Ньютона**:
   - Преимущества: быстрая сходимость, эффективное использование информации о кривизне функции
   - Недостатки: требует вычисления градиента, чувствителен к начальному приближению

## Заключение

Овражные функции представляют особую сложность для методов оптимизации. Для данной конкретной задачи метод Ньютона показал значительное преимущество по сравнению с методом Нелдера-Мида благодаря использованию информации о градиенте и кривизне функции.

Для овражных функций рекомендуется:
1. Использовать методы, учитывающие кривизну (Ньютон, квазиньютоновские методы)
2. Применять предобусловливание для улучшения обусловленности задачи
3. Рассмотреть возможность масштабирования переменных для уменьшения различий в масштабах градиента по разным направлениям

Оптимальное решение для данной функции соответствует точке, где обе линейные комбинации $(9x+8y-5)$ и $(5x+2y-1)$ одновременно близки к нулю.
